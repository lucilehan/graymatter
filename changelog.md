# Changelog

## 0.1.0 — Initial Release

- Interactive 3D brain visualization with WebGL raymarching
- Halftone dot-art rendering with sparkle effects
- Six memory categories mapped to anatomical brain regions
- Add, forget, import, and export memories
- AI-powered categorization via Anthropic API
- Scroll to zoom, drag to rotate, click regions to explore
- Auto-rotate to region on category selection
- Resizable panel divider
- Persistent JSON storage in workspace
- Automated publishing via GitHub Actions
